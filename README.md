# linhas_de_separacao

Description. 
The package linhas_de_separação is used to:
	linhas_simples:
		- print simple line
	linhas_alternadas:
		- print alternate lines

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install linhas_de_separacao

```bash
pip install linhas_de_separacao
```
